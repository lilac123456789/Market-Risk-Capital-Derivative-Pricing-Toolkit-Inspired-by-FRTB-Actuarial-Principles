# Actuarial–Finance Module

This module blends actuarial modeling with quantitative finance concepts:

### 1. Survival Model with Interest Rates
- Life-contingent valuation
- Continuous survival function
- Discounting at stochastic or constant interest rates

### 2. Credit Migration Using Markov Chains
- Transition matrix modeling
- n-step transition probabilities
- Expected rating distribution

### 3. Expected Loss and PD/LGD Modeling
- Actuarial-style loss estimation
- Linking survival probabilities with credit risk metrics
