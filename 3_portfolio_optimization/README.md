# Portfolio Optimization Module

This module implements classic portfolio optimization techniques used in quantitative finance:

### 1. Markowitz Efficient Frontier
- Mean-variance optimization
- Covariance matrix modeling
- Frontier plotting (weights vs. risk-return)

### 2. Minimum Variance Portfolio (MVP)
- Closed-form solution using inverse covariance matrix

### 3. Risk Parity Portfolio
- Iterative risk budgeting solver
- Equal risk contribution (ERC)
